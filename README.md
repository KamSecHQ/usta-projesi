# Usta — Türkiye'nin Yazılım Çarşısı

Doğrulanmış Türk yazılımcılarla işletmeleri bir araya getiren bir pazaryeri konsepti. React ile sıfırdan geliştirildi.

## Özellikler
- 🏠 Çok bölümlü anasayfa (Hero, Süreç, Ustalar, Neden Usta)
- 🔍 Aranabilir ustalar listesi
- 📝 Yazılımcı ve iş veren başvuru formları
- 🔐 Supabase Auth ile kayıt/giriş sistemi
- 📊 Korumalı admin paneli (başvuru yönetimi + filtreleme)
- 🗄️ Supabase (PostgreSQL) veritabanı entegrasyonu, Row Level Security ile

## Kullanılan Teknolojiler
- React + Vite
- React Router
- Tailwind CSS
- Supabase (Auth + Database)

## Kurulum
\`\`\`bash
npm install
npm run dev
\`\`\`
